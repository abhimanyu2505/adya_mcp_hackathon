## 🔐 Credential JSON Payload
Example payload format for sending credentials to the MCP Server which going to be use it in Client API paylod:
```json
{
  "GITHUB": {
    "GITHUB_PERSONAL_ACCESS_TOKEN": "your-github-personal-access-token"
  }
}
```
